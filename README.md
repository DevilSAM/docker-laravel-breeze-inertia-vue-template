## Template

Это шаблон для проектов

### Используется докер с отдельными контейнерами:
- php
- node
- mysql
- nginx
- rabbitmq

### Особенности
- Laravel framework
- Laravel breeze (авторизация)
- Inertia с Dark mode
- Vue 3
- Prettier + ESLint
- Vite

### Запуск:
```
docker compose build
docker compose up -d
```
